函数相关的测试

  • js函数直接作为java函数参数
    代码：
    demo.fun.jsFun(function(){ alert('js函数直接作为java函数参数'); });

  • js函数作为json对象的内部变量
    代码：
    demo.fun.jsFun({ 'callback':function(){ alert('js函数作为json对象的内部变量'); } });

  • js函数作为json数组的元素
    代码：
    demo.fun.jsFunArray([function(){ alert('js函数作为json数组的元素'); }]);

  • js函数作为json对象的json数组的元素
    代码：
    demo.fun.jsFunArray({ 'array':[function(){ alert('js函数作为json对象的json数组的一个元素'); }] });

  • 直接传递java函数对象
    代码：
    demo.fun.sendJavaFun(function(javaCall){ javaCall('直接传递java函数对象'); });

  • java函数对象作为json对象的一个变量
    代码：
    demo.fun.sendJavaFunInJson(function(obj){ obj.callback('java函数对象作为json对象的一个变量'); });
